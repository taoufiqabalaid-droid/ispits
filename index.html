Guide de configuration Google Sheets

1

Créer un Google Sheet

Allez sur sheets.google.com et créez une nouvelle feuille. Ajoutez ces en-têtes dans la première ligne :

Date | Nom | Prénom | Email | Téléphone | Naissance | Genre | Fichier Certificat | Durée Certificat | Date Début | Date Fin | Type Certificat
2

Ouvrir Apps Script

Dans votre Sheet, allez dans Extensions → Apps Script. Un éditeur de code s'ouvrira.

3

Coller le code

Supprimez tout le contenu existant et collez ce code :

function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    var data = JSON.parse(e.postData.contents);

    sheet.appendRow([
      new Date().toLocaleString('fr-FR'),
      data.lastName,
      data.firstName,
      data.email,
      data.phone,
      data.birthDate,
      data.gender,
      data.certFileName,
      data.certDuration,
      data.certStart,
      data.certEnd,
      data.certType
    ]);

    return ContentService
      .createTextOutput(JSON.stringify({status: 'success'}))
      .setMimeType(ContentService.MimeType.JSON);

  } catch(error) {
    return ContentService
      .createTextOutput(JSON.stringify({status: 'error', message: error.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  return ContentService
    .createTextOutput(JSON.stringify({status: 'connected'}))
    .setMimeType(ContentService.MimeType.JSON);
}
4

Déployer

Cliquez sur Déployer → Nouveau déploiement :

  • Type : Application web
  • Exécuter en tant que : Moi
  • Accès : Toute personne

Cliquez sur Déployer, autorisez l'accès, puis copiez l'URL de l'application web.

5

Coller l'URL ici

Cliquez sur l'icône ⚙️ en haut à droite de cette page, collez l'URL obtenue, puis cliquez Enregistrer et Tester.
